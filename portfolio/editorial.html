The Project

A studio fashion editorial session in Los Angeles, August 2025. The project built a clean, commercially grounded editorial range using controlled studio conditions. Wardrobe included tights and Guess pieces, styling that sat clearly within the editorial category without overreaching. Model Julia.

This type of work is designed to function as strong portfolio material: a consistent visual direction across multiple looks, executed in a studio setting that keeps the focus on the model and the wardrobe.

The studio setting kept the framing deliberate throughout. Studio editorial work like this (a defined wardrobe direction, controlled light, and a clear brief) is how a model's portfolio range gets built between commercial jobs.

Agencies need current imagery that shows range across categories. A well-executed studio session with a specific wardrobe direction gives them that: consistent, portfolio-grade work that holds up in casting comparisons.
